1. Перейти в необходимую директорию cd 
   git clone <url>
   Ссылку копируем в SSH.

2. Создать файлю.txt
   git add файл.txt
   git commit -m "Коммит файл.txt"
   git log
   git push

3. git fetch

4. git status - отображает состояние наших файлов,
   показывет файлы которое не отслеживаются GIT
   git log — история коммитов
   git remote -v - показывет репозиторий на гитхабе retch, push.
